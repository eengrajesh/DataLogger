===============================================================================
BUTTON FUNCTIONS (HOW THEY START THE PROGRAM):
===============================================================================

GREEN START BUTTON (GPIO 17):
- Single Press: START temperature logging
- Program will begin reading thermocouples and saving data
- SYSTEM LED will turn GREEN when running

RED SHUTDOWN BUTTON (GPIO 27):
- Single Press: STOP temperature logging  
- Long Press (5 sec): Shutdown entire Pi system
- ERROR LED will flash RED during shutdown

BLUE EXPORT BUTTON (GPIO 22):
- Single Press: Export current data to CSV file
- Data saved to /data/exports/ folder
- NETWORK LED will blink BLUE during export

YELLOW WIFI BUTTON (GPIO 23):
- Single Press: Reset WiFi connection
- Long Press: Enter WiFi setup mode
- LOGGING LED shows YELLOW when in WiFi mode

===============================================================================
LED STATUS INDICATORS:
===============================================================================

GREEN SYSTEM LED: System operational status
- ON: System running normally
- SLOW BLINK: Starting up
- OFF: System stopped

RED ERROR LED: Error and alert status  
- ON: Critical error occurred
- FAST BLINK: Warning condition
- OFF: No errors

BLUE NETWORK LED: Network connectivity
- ON: Internet connected
- SLOW BLINK: WiFi connected, no internet
- FAST BLINK: Connecting to WiFi
- OFF: No network connection

YELLOW LOGGING LED: Data logging status
- ON: Currently logging temperature data
- SLOW BLINK: Logging paused
- OFF: Not logging

===============================================================================
TESTING PROCEDURE:
===============================================================================

1. BEFORE POWERING ON:
   ‚úì Double-check all ground connections
   ‚úì Verify no short circuits between power and ground
   ‚úì Ensure LEDs connected with correct polarity (+/-)
   ‚úì Confirm resistors in series with LEDs

2. POWER ON PI5 AND TEST:
   ‚úì All 4 LEDs should briefly flash during startup
   ‚úì Press each button - should see response in terminal/web interface
   ‚úì GREEN button starts logging (YELLOW LED turns on)
   ‚úì RED button stops logging (YELLOW LED turns off)

3. TROUBLESHOOTING:
   - LED not lighting: Check polarity and resistor connection
   - Button not working: Check ground connection
   - Multiple issues: Check all power connections (3.3V/GND)

SAFETY NOTES:
===============================================================================

- All components use safe low voltages (3.3V max)
- No risk of electrical shock
- Pi5 GPIO pins are 3.3V tolerant
- Total current draw: ~100mA (very low)
- No external power supply needed
- Components are beginner-friendly
